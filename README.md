# Nextwork-projects

This repository contains the projects completed as part of the Nextwork exercises. The projects cover a series of DevOps and DevSecOps labs and exercises, including CI/CD pipelines, infrastructure as code, containerization, monitoring, security hardening, and automated testing.

Contents:
- DevOps exercises: CI/CD pipelines, Docker, Kubernetes, infrastructure as code (Terraform/Ansible), monitoring and logging.
- DevSecOps exercises: security scanning, vulnerability remediation, secure pipeline practices, secrets management, and compliance checks.

Each project folder includes setup instructions, configuration files, and notes from the exercises.
